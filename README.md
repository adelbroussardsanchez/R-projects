# R Projects

## Overview
This folder contains R-based data analysis projects that focus on real-world questions and applied statistical analysis. Each project includes data cleaning, exploratory analysis, visualizations, and modeling, with an emphasis on clearly explaining results rather than only reporting statistics.

The projects in this folder are structured as standalone analyses and can be reviewed independently.

---

## Projects Included

### Resume Callback Analysis
An analysis of a resume audit dataset to examine whether race and gender affect the likelihood of receiving a callback after applying for a job. This project focuses on exploratory analysis, model building, and validation to understand which factors matter most.

**Key focus areas:**
- Data cleaning and preparation  
- Visual comparison of callback rates across groups  
- Predictive modeling and model validation  

---

### Gun Control and Gun-Related Crime Analysis
A statistical analysis exploring the relationship between gun control laws and gun-related crime rates across U.S. states. This was an early project aimed at testing a clear hypothesis using data and building foundational analysis skills.

**Key focus areas:**
- Exploring relationships between laws and crime rates  
- Visual and numerical analysis  
- Regression modeling and interpretation   

---

## Tools Used
- **Language:** R  
- **Common Packages:** openintro, ggplot2, leaps, AER, caret
- **Skills Demonstrated:** data cleaning, visualization, statistical analysis, regression
